UK Banking Dashboard
Tableau

This project involved creating a Bank Dashboard using Tableau with UK banking data from Scotland, Northern Ireland, England, and Wales. It includes visualizations like customer age histogram, account balance histogram, gender distribution, and job details of account owners. The dashboard features interactive slicers for age, account balance, and gender, along with cross-filtering for dynamic data exploration. This enables users to gain valuable insights into banking trends and customer demographics.

Supermarket Sales Dashboard
Power BI

This is one of my first projects using Power BI. Kudos to Lean Excel Solutions for the original content.

This project involved creating a Sales Dashboard using Power BI to analyze supermarket sales data. It features key visualizations such as cards for total sales, profit, top product, and top category, along with stacked column charts, bar charts, pie charts, area charts, and tree charts. The dashboard includes dynamic filtering with slicers for month, year, sales type, and payment mode, providing interactive visualizations and cross-filtering for detailed sales performance analysis.
